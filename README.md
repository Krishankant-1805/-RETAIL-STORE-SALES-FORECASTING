# 📦 Retail Store Inventory Forecasting

## 📌 Project Overview

This project predicts inventory demand using historical retail store data. It includes data cleaning, exploratory data analysis (EDA), machine learning, and an interactive Power BI dashboard to help improve inventory management.

---

## 🎯 Objectives

- Analyze inventory data
- Identify sales and demand patterns
- Forecast inventory demand
- Build an interactive dashboard

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI
- Git & GitHub

---

## 📂 Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Machine Learning Model
5. Model Evaluation
6. Power BI Dashboard

---

## 📊 Dashboard Features

- Total Inventory
- Total Units Sold
- Average Demand Forecast
- Average Price
- Units Sold Over Time
- Inventory by Category
- Region-wise Inventory
- Top 10 Products
- Interactive Filters

---

## 📈 Machine Learning

Model Used:
- Random Forest Regressor

Evaluation Metrics:
- MAE
- RMSE
- R² Score
